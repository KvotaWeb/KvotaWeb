## About the Original (Minecraft 4k & Classic Versions)

This is a port of **Minecraft 4k**, Markus "Notch" Persson's entry for the Java 4K Game Programming Contest (July 2009).

This legendary 4KB applet is what directly spawned Minecraft as we know it. 

It wasn't the first block prototype though. Here's the full chronology of the very first, classic versions of Minecraft: 

| Version | Date | What it was |
|----------|-----|------------|
| **RubyDung** | 16 May 2009 | Very first block prototype. No terrain generation, just a small test with textured blocks and a player. The direct predecessor to Minecraft |
| **rd-132211** (Pre-Classic) | 13 May 2009 | The infamous "Cave Game". Extremely barebones, first test of a randomly generated cave. This is the absolute first build of what would become Minecraft |
| **Minecraft 4k** | 31 May 2009 - 13 July 2009 | The 4KB Java Applet. Random terrain, trees, water, sand, lava, TNT, placing/destroying blocks and a very basic save. The version this port recreates |
| **Minecraft Classic** | 16 May 2009 (First Public) - 10 November 2009 | The browser Java applet. Multiplayer support, terrain generation, many more block types. This is the version most people remember when they hear "Minecraft Classic" |
| **Survival Test** | 16 September 2009 | First version with health, mobs (spiders, skeletons, zombies), survival mechanics and dropping items |
| **Indev** | 23 December 2009 | First "in development" version. Huge maps, crafting, tools, time of day and the beginning of the game we know today | 
| **Infdev** | 27 February 2010 | Infinite terrain generation. |


## Notch's other classic games

Long before (and during) Minecraft, Notch created tons of brilliant 4K games and Ludum Dare entries. 

Many of these share the exact same creative, procedural, minimalist style. 

Here are some of the classics: 

| Game | Year | About |
|----------|---|---|
| **Left 4K Dead** | 2008 | 4K Zombie shooter. Very famous 4K entry |
| **Hunters4K** | 2008 | 4K Top down shooter |
| **t4xi** | 2008 | 4K Action game. |
| **Breaking the Tower** | 2008 | 4K Puzzle/platformer |
| **Bunny Press** | 2008 | 4K Arcade game |
| **Metagun** | 2008 | Brilliant 2D platformer. Arguably his best 4K game (and one of the greatest 4K games ever made!) |
| **Prelude of the Chambered** | Ludum Dare 21 (28 August 2011) | Dungeon crawler. Procedurally generated dungeon. Beautiful, minimal and legendary |
| **Minicraft** | Ludum Dare 22 (17 December 2011) | A full, complete 2D top-down survival game. Made in 48 hours. Absolutely legendary. |
| **Legend of the Chambered** | Ludum Dare 22 Jam (19 December 2011) | Sequel to Prelude of the Chambered. Even better dungeon crawling goodness! |
| **Shambles** | Ludum Dare 23 (20 April 2012) | Tiny adventure game |
| **Spewer** | Ludum Dare 23 (21 April 2012) | Brilliant physics based platformer. One of his most creative games! |


## Requirements

Absolutely nothing. 

Just a modern web browser with HTML5 Canvas support. 

Works on Chrome, Firefox, Edge, Brave, Opera and Safari. (Anything from ~2012 onward will run it perfectly)


## Download & Run Locally

This project requires no build tools. It's pure HTML and JavaScript. 

**Option 1: Download ZIP**

1. Download: [KvotaWeb-main.zip](https://github.com/Artik9nx/KvotaWeb/archive/refs/heads/main.zip)
2. Extract it 
3. Open `index.html` in your web browser
